# KillAll3k - 全能办公效率工具
KillAll3k（要你命三千）是一款基于 **PyQt6** 开发的超级办公效率工具，集**软件启动、文件搜索、剪贴板历史管理、屏幕截图**四大核心功能于一体，通过轻量化操作、全局快捷键、智能识别等特性，全面提升办公与操作效率，支持现代化界面与个性化配置。

## 核心特性总览
- 基于 PyQt6 构建，轻量化运行，界面现代化
- 开机自启动、系统托盘常驻，后台无感运行
- 明暗主题自动跟随系统，支持手动切换
- 集成 AI 能力，可对剪贴板/截图内容进行 AI 对话
- 全局快捷键自定义，所有核心功能一键唤醒

---

## 一、全局基础功能
1. **系统托盘运行**
   - 默认启动后最小化至系统托盘，关闭窗口不退出程序
   - 托盘图标右键菜单快速打开设置面板
2. **个性化主题设置**
   - 现代化明暗双主题，默认**自动跟随系统主题**
   - 支持手动强制切换亮色/暗色模式
3. **系统配置项**
   - 开机自启动（可开关）
   - 启动通知（默认关闭）
   - 管理员权限运行（默认开启）
4. **AI 模型配置**
   - 自定义各类 AI 大模型 API 接口
   - 支持对**选中的剪贴板记录**/**当前截图**发起 AI 对话

---

## 二、软件启动器
快速检索、一键启动本地应用，替代传统开始菜单，极致提速。
1. **全局快捷键唤醒**
   - 默认快捷键：`Alt + Space`
   - 支持用户自定义唤醒快捷键
2. **智能搜索能力**
   - 支持应用名称**模糊搜索**，输入关键词秒级匹配
   - 自动扫描系统已安装软件
   - 支持自定义添加软件扫描目录，适配绿色软件
3. 一键启动匹配结果，支持回车快速打开首个应用

---

## 三、文件搜索器
类 Everything 极速文件检索，支持高级匹配与排序。
1. **核心搜索功能**
   - 文件名**模糊匹配**，极速检索本地文件
   - 支持通配符搜索（`*`、`?`），精准定位文件
2. **结果管理**
   - 多维度排序：文件名、修改时间、文件扩展名
   - 支持直接打开文件/所在文件夹

---

## 四、剪贴板历史管理
全方位管理剪贴板数据，解决复制粘贴丢失、重复操作问题，支持多类型内容。
1. **全局唤醒**
   - 默认快捷键：`Win + V`
   - 兼容系统原生 Win+V 占用，可自由恢复/切换
   - 支持自定义显示/隐藏剪贴板窗口快捷键
2. **安全监听**
   - 仅监听系统 `Ctrl+C` 复制操作
   - **不修改系统原生剪贴板内容**，安全无侵入
3. **记录管理**
   - 支持设置历史记录上限：100/500/1000条/无上限（默认）
   - 兼容**文本、图片、文件**多类型记录
   - 分类筛选展示，快速定位目标内容
4. **编辑与操作**
   - 文本修改、图片标注编辑
   - 合并复制、合并粘贴、批量收藏、批量删除
   - 支持记录排序、置顶、收藏、搜索、预览
5. **高级粘贴**
   - 带格式粘贴 / 无格式纯文本粘贴
6. **在线搜索**
   - 调用默认浏览器在线搜索剪贴板内容
   - 可自定义/切换搜索引擎

---

## 五、屏幕截图工具
集智能截图、长截图、标注、OCR、翻译、贴图于一体，满足全场景截图需求。
1. **智能截图模式**
   - 智能窗口识别：自动识别界面元素边界，精准截取
   - 支持全屏截图、窗口截图、焦点窗口截图
2. **长截屏（滚动截图）**
   - 支持滚动截取长页面、长文档、长网页
3. **截图标注工具**
   - 截图完成默认进入标注模式
   - 工具集：选择、画笔、形状、箭头、文字、序号、马赛克、高亮、水印
4. **OCR 文字识别**
   - 基于 **PaddleOCR-v5** 实现高精度文字提取
5. **OCR 翻译功能**
   - 支持识别结果一键翻译
   - 可配置翻译接口：有道翻译、Google翻译、自定义AI模型翻译
6. **保存与导出**
   - 自动保存截图至剪贴板历史（默认）
   - 支持手动保存到本地指定路径
7. **屏幕贴图功能**
   - 截图固定悬浮在屏幕顶层
   - 支持移动、缩放、调整透明度、重新标注、保存、删除
8. **快捷键配置**
   - 默认截图快捷键：`Ctrl + Alt + A`
   - 支持用户自定义修改

## 技术栈
- 开发框架：**PyQt6**
- OCR 引擎：PaddleOCR-v5
- 运行环境：Windows 系统
- 核心能力：全局热键、系统托盘、文件检索、剪贴板监听、屏幕捕获、AI 接口集成
